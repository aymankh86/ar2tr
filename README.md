# ar2tr

## Run locally

1. create virtualenv and activate it
  ```sh
  virtualenv venv
  source ./venv/bin/activate
  ```
2. install the requirements
  ```sh
  pip install -r requirements.txt
  ```
3. run
  ```
  python manage.py runserver
  ```
4. open from here [localhost:8000](http://localhost:8000)
